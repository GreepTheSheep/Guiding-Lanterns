Introducing the Tangled screencap command with The Guiding Lanterns

Enjoy quality screenshots from the movie (and the series as well)
!scr movie 1:30:00
!scr S01E05 20%
Try these commands out on our Discord server

The Guiding Lanterns' Dev. Server

Using the command

!scr <video_id> <timestamp>

The <timestamp> may be a number (in seconds), a percentage(eg. 50%) or in a format hh:mm:ss.xxx (where hours, minutes and milliseconds are optional)
